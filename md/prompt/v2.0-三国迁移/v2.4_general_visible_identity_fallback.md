# v2.4 武将可见身份 fallback 收敛提示词

## 目标

- 继续推进“注重武将、道路、交战”的迁移，把武将影响、计划军令、武将层复核事件和军机面板中可能露出的 raw `generalId`、`RegionId`、`FrontZoneId` 收敛为中文展示名或中文占位。
- `GeneralInfluence` 的道路/交战日志与预览优先使用武将姓名；缺少姓名快照但仍有内部 id 时显示“未命名武将”，完全无分配时保持“未分配/未任命”语义。
- `TurnManager` 的君主、太守、军师、武将层诊断和事件摘要使用郡县/防区展示名，缺失时显示“未知郡县 / 未知防区”。
- `AgentPanelView` 和 `AppContainer` 的武将记录、计划军令摘要、交战候选和接近预览不把内部 id 当作玩家文案 fallback。

## 范围

- `WWIIHexV0/Rules/GeneralInfluence.swift`
- `WWIIHexV0/App/AppContainer.swift`
- `WWIIHexV0/UI/AgentPanelView.swift`
- `WWIIHexV0/UI/RootGameView.swift`
- `WWIIHexV0/Turn/TurnManager.swift`
- `WWIIHexV0/Agents/GeneralAgent.swift`
- `WWIIHexV0/Agents/AgentDecision.swift`
- 相关文档：`README.md`、`md/flow/flow.md`、`md/flow/flowchart.md`、`md/prompt/README.md`、`update_log.md`

## 非目标

- 不改变 `generalId`、`RegionId`、`FrontZoneId`、`ZoneDirective`、`GeneralDecisionRecord`、Codable、JSON schema 或记录 id。
- 不隐藏或重写完整调试 JSON；调试 JSON 仍可保留 raw 字段供审计。
- 不改变 `GeneralInfluence` 道路机动、攻击、防御数值。
- 不改变 `WarCommandExecutor`、`RuleEngine`、移动、攻击、占领、道路修缮或外交敌对判断。
- 不运行本机 Xcode / XCTest / 模拟器 / Probe / Smoke / Stage Regression / Dynamic Theater Regression / Full。

## 架构依据

```text
GeneralAssignment / GeneralDecisionRecord
  -> GeneralInfluence movement/combat summary
  -> TurnManager diagnostics / AppContainer previews / AgentPanelView
  -> EventLog / GeneralCommandPanelView / UnitInspectorView / AgentPanelView
```

- 日志、诊断和预览是只读展示层，不是状态权威。
- 底层身份字段继续保留，用于 Codable、排序、记录关联和调试审计。
- 玩家可见 fallback 应优先表达“未知/未命名/未分配”，而不是泄漏内部 id。

## 并发安排

- 子 Agent A 只读审计 `TurnManager.swift` 的武将名、防区名和郡县名 raw fallback。
- 子 Agent B 只读审计 `GeneralInfluence.swift`、`AppContainer.swift`、`AgentPanelView.swift` 的武将道路/交战/计划军令 raw fallback。
- 主 Agent 统一整合，避免多个子 Agent 同时改同一文件。

## 验收标准

- 武将道路/交战日志和预览不再输出 `attacker`、`defender`、`general` 或 raw `generalId` fallback。
- 计划军令摘要、武将层复核事件、军机面板武将记录和层级诊断不再把 raw `RegionId` / `FrontZoneId` 当作默认展示名。
- 调试 JSON、记录 id、排序、Codable 字段和规则行为保持兼容。
- 本机只跑轻量检查；重验证交给 `origin/main` 的 GitHub Actions。

## 风险

- 本轮不隐藏完整调试 JSON，因此开发审计区域仍可能显示 schema 内部字段名；这属于调试信息策略，后续可单独切片。
- 本轮不做 UI 截图或模拟器验收，中文占位的实际换行和宽度需后续人工或授权运行环境确认。
