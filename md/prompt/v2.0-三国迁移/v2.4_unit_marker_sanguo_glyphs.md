# v2.4 地图兵牌三国兵种符号与姿态中文化

## 目标

- 继续推进地图第一视口的三国化，把 `UnitNode` 中可见的 NATO APP-6 兵牌图形和 `R/H` 姿态标记替换为三国兵种 glyph 与中文短姿态。
- 让玩家在地图上直接看到步、骑、弓、械、卫、舟、轻等兵种符号，以及 `退/守` 姿态提示。

## 范围

- 只改 `WWIIHexV0/SpriteKit/UnitNode.swift` 的兵牌显示。
- 同步更新 README、核心流程文档、流程图、prompt 索引和 `update_log.md`。
- 保持 `Division` 数据、`RetreatMode` rawValue、移动/战斗/撤退规则、地图节点布局和 `BoardScene` 实例化流程不变。

## 禁止项

- 不改 `Division`、`ComponentType`、`RetreatMode` 的 Codable/rawValue。
- 不改 `BoardScene` 交互、选中逻辑、补给标记、堆叠标记或玩家微操金色圈。
- 不引入新图片资源、asset catalog 或自定义字体。
- 不运行 Xcode / XCTest / 模拟器 / Probe / Smoke / Stage Regression / Dynamic Theater Regression / Full。

## 架构依据

- `UnitNode` 已有私有 `Division.markerCode`，能输出 `械/骑/弓/卫/舟/轻/步`。
- `RetreatMode.shortDisplayCode` 已在核心模型中提供 `退/守`，可直接复用。
- `BoardScene` 只负责创建 `UnitNode`，不需要改动实例化路径。

## 实现步骤

1. 将 `addNATOSymbol` 替换为中心 `addUnitKindGlyph` 文本 glyph。
2. 删除 NATO APP-6 注释和图形绘制 helper。
3. `markerReadinessText` 改用 `retreatMode.shortDisplayCode`。
4. 删除 `UnitNode` 私有 `RetreatMode.markerCode`。
5. 更新阶段 prompt、prompt 索引、README、核心流程文档和更新日志。
