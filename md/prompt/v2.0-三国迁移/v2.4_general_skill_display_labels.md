# v2.4 武将技能中文展示提示词

> 本阶段承接 `v2.4_general_tactic_shaping.md`、`v2.4_general_panel_tactic_audit.md` 和 `v2.4_general_influence_panel_summary.md`。执行前必须先读 `AGENTS.md`、`update_log.md`、`md/flow/flow.md`、`md/flow/flowchart.md`、`md/test/test.md` 和相关源码。

## 目标

- 让玩家可见的武将技能不再显示 `rapid_exploitation`、`logistics` 等 raw id。
- 保留 `GeneralData.skills` / `GeneralAssignment.skills` 的 raw id，避免破坏 JSON、Codable、规则判断和旧数据。
- 在武将档案、武将军令面板和武将复核 rationale 中显示三国化中文技能标签。

## 范围

- 主要文件：
  - `WWIIHexV0/Agents/GeneralRegistry.swift`
  - `WWIIHexV0/Agents/GeneralAgent.swift`
  - `WWIIHexV0/UI/GeneralCommandPanelView.swift`
  - `WWIIHexV0/UI/GeneralProfileView.swift`
- 文档文件：
  - `README.md`
  - `md/flow/flow.md`
  - `md/flow/flowchart.md`
  - `md/prompt/README.md`
  - `update_log.md`
- 不修改 JSON schema、不改 `GeneralInfluence`、`MovementRules`、`CombatRules`、`WarCommandExecutor` 或 `RuleEngine`。

## 实现要求

- 在武将数据层附近增加轻量展示 helper，例如 `GeneralSkillDisplay.displayName(for:)`。
- 常见技能至少覆盖当前 `sanguo_generals.json` 和旧 `generals.json` 中出现的道路、交战、守备、预备、攻城、协同类技能。
- 未登记的新技能使用安全 fallback 展示，不能影响规则执行。
- `GeneralCommandPanelView` 和 `GeneralProfileView` 使用中文技能标签。
- `GeneralAgent` 的 rationale 继续保留技能摘要，但用中文标签替换 raw id。

## 禁止项

- 不允许把中文标签写回 `skills` 数据字段。
- 不允许改变技能 raw id 的排序、Codable 字段、规则匹配条件或战斗/道路计算。
- 不允许新增本机 Xcode build/test、模拟器、Probe、Smoke、Stage Regression、Dynamic Theater Regression、Full 或性能测试。
