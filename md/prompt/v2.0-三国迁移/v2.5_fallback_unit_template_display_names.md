# v2.5 fallback 军队模板展示名三国化提示词

## 目标

继续推进 v2.5 发布级可见文案收口，把旧 fallback `WWIIHexV0/Data/unit_templates.json` 中可能被缺省资源路径加载到的英文军队模板展示名改为中文三国语义，避免 `Panzer Division`、`Motorized Division`、`Infantry Division`、`Artillery Division`、`Anti-Tank Division`、`Garrison Division` 在 fallback 面板、日志或编辑器诊断中透出。

## 范围

- 只允许修改 `WWIIHexV0/Data/unit_templates.json` 的 6 个 `displayName`。
- 建议映射：
  - `panzer_division` -> `甲骑军`
  - `motorized_division` -> `疾行军`
  - `infantry_division` -> `步卒营`
  - `artillery_division` -> `器械营`
  - `anti_tank_division` -> `破甲营`
  - `garrison_division` -> `守备营`
- 更新 `md/prompt/README.md` 和 `update_log.md` 记录本轮数据展示名收口。

## 非目标

- 不修改模板 `id`、`maxHP`、`components`、component `type`、权重或 JSON schema。
- 不修改默认 `sanguo_unit_templates.json`。
- 不修改 `DataLoader` 的优先加载顺序、旧 fallback 资源名、`loadArdennes*` API、规则、AI、MapEditor 导出结构或命令管线。
- 不清理历史 prompt / old 记录中的旧二战示例文本。

## 架构依据

- 默认模板加载先读 `sanguo_unit_templates.json`，旧 `unit_templates.json` 只作缺资源 fallback。
- `DataLoader.makeDivisions()` 建立 `Division` 时读取模板 `components` 和 `maxHP`，军队实例名来自剧本 unit `name`，不依赖 fallback 模板 `displayName` 做规则判断。
- 既有测试只断言模板 `id` 集合和权重，不断言这 6 个展示名。
