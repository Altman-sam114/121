# v2.4 武将接敌配对箭头中文化

## 目标

继续推进武将、道路和交战可见审计闭环，把武将军令面板“道路与交战”摘要中的接敌配对从 ASCII `->` 改为中文 UI 已使用的 `→`，保持玩家可见交战关系与计划军令源→目标显示口径一致。

## 范围

- 修改 `WWIIHexV0/App/AppContainer.swift` 中 `engagementPairingText(...)` 的展示文本。
- 同步 `md/prompt/README.md` 与 `update_log.md`。

## 非目标

- 不改变 `GeneralInfluence`、`CombatRules`、`MovementRules`、`WarCommandExecutor`、`RuleEngine`、接敌配对排序、攻防修正、距离计算或可见性规则。
- 不修改 JSON schema、Xcode project、测试文件、地图数据或旧兼容 rawValue。
- 不运行本机 Xcode / XCTest / 模拟器 / Probe / Smoke / Stage Regression / Dynamic Theater Regression / Full。

## 实现步骤

1. 将 `AppContainer.engagementPairingText(...)` 输出的 `attacker -> defender` 改为 `attacker → defender`。
2. 保留攻击/防御视角、修正值、距离、排序和候选筛选不变。
3. 更新阶段 prompt 索引和版本日志。

## 轻量检查
